## 운영 규칙
- 임시 우회는 만료일과 제거 조건 필수
- 스프린트마다 상위 부채 1개 이상 상환
- `open` 상태 30일 초과 항목은 기술리더 리뷰 대상
- 신규 부채 등록 시 영향도/상환비용/우선순위를 함께 기록
