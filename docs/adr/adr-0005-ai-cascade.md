# ADR-0005 AI 평가 3단계 캐스케이드 채택

- 상태: accepted
- 작성일: 2026-02-16
- 의사결정자: AI/Platform Team

## 배경/문제
모든 제출을 고비용 모델로 평가하면 비용이 급증하고, 저비용 모델만 사용하면 품질이 불안정해진다.

## 선택지
1. 단일 저비용 모델
2. 단일 고성능 모델
3. 다단계 캐스케이드(Tier1/2/3)

## 결정
기본은 저비용 모델로 처리하고, 신뢰도/난이도/불일치 조건에서 상위 모델로 승급하는 3단계 캐스케이드를 채택한다.

## 근거
- 비용과 품질의 균형 최적화
- 프리미엄/고난도 시나리오 대응 가능
- 캐시/배치 전략과 결합 시 운영 효율 상승

## 결과/영향
- 장점: 비용 절감, 품질 방어, 확장성
- 단점: 라우팅 로직/모니터링 복잡도 증가
- 후속 작업: 승급 규칙 실험(A/B), 품질-비용 대시보드 운영
